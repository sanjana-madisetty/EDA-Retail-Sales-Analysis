# EDA on Retail Sales Data

## Project Overview

This project performs Exploratory Data Analysis (EDA) on a retail sales dataset to identify sales trends, customer behavior, and product performance.

## Objectives

* Analyze retail sales data
* Identify sales patterns and trends
* Understand customer demographics
* Visualize important insights
* Support business decision-making

## Tools Used

* Python
* Pandas
* Matplotlib
* Seaborn

## Dataset Features

* Transaction ID
* Date
* Customer ID
* Gender
* Age
* Product Category
* Quantity
* Price per Unit
* Total Amount

## Analysis Performed

* Data Inspection
* Missing Value Check
* Descriptive Statistics
* Monthly Sales Analysis
* Product Category Analysis
* Customer Demographics Analysis
* Correlation Analysis

## Visualizations

* Monthly Sales Trend
* Product Category Sales
* Gender Distribution
* Age Distribution
* Correlation Heatmap

## Conclusion

The analysis provided insights into sales performance, customer demographics, and purchasing behavior. These insights can help businesses improve marketing strategies and inventory management.
